About KSMS

The Kubernetes Security Monitoring System (KSMS) is designed to provide real-time visibility into the security posture of Kubernetes clusters.

Project Information

  • Project Name: Kubernetes Security Monitoring System
  • Author: Galya Dodova (FN 45616)

Technologies Used

  • Backend: Golang, Gorilla Mux, JWT, bcrypt
  • Frontend: Golang html/template, Vue.js, Bootstrap 5
  • Monitoring: Prometheus Metrics (planned), SSE for real-time alerts
  • Kubernetes Integration: Client-go (planned)

Features

  • Real-time security alert monitoring via Server-Sent Events (SSE).
  • Management of multiple Kubernetes clusters and their health metrics.
  • Definition and enforcement of security policies across namespaces.
  • Role-Based Access Control (RBAC) for Administrators, Security Analysts, and Students.
  • Comprehensive incident reporting and historical data tracking.
